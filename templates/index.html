A Flask Weather App Demo

A simple Weather Flask App Demo. Unfournately sharing Python applications is combersome. I wanted to learn Flask to be able to have an additional tool to share Python functionality with less technical users. In this Demo application I use Openweathermap.org's free "One Call API" in combination with the "Current Weather Data" API. Additionally, designed a basic webpage utilizing bootstrap.
